Free data

To make a map, you need digital data. Fortunately, there's tons of it available for free.

The format you'll mostly want is called the "Shapefile." (It's actually a group of files with the same name but different extensions.)

Some sources:

  • naturalearthdata.com/
  • census.gov/geo/maps-data/
  • gisdata.mn.gov
  • opendata.minneapolismn.gov
  • hennepin.us/gisopendata

Next steps

I've demonstrated some techniques for making simple but attractive maps in QGIS. But it can do a lot more than I've shown you today.

For example, QGIS can:

  • Match up basic geography with data such as election results, test scores or pollution levels
  • Find every house located within the half-mile blast zone of an oil train route
  • Calculate the percentage of every Minnesota county's land area made up of lakes
  • Calculate which latitude/longitude points fall inside which of a given set of boundaries
